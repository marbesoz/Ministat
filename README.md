File sorgente, file di installazione e ISO dell'installazione di Ministat, programma di statistica scritto nei primi anni '90 con Visual Basic, gira anche su Windows 11, escluso l'help che però funziona se si installa Ministat su una macchina virtuale Windows 2000 o Windows Xp realizzata con VirtualBox.  

Il sorgente contiene il progetto VB ed è stato verificato che su entrambe le macchine citate viene correttamente compilato con Visual Basic 5.

Orientato al campo biomedico, Ministat include anche la statistica bayesiana (sensibilità, specificità, valore predittivo e curve ROC).
