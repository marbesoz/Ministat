Sorgente, eseguibile e ISO dell'installazione di Ministat, programma di statistica scritto con una delle prime versioni di VisualBasic e ancora funzionante (eccetto l'help) con Windows 11. Funziona anche l'help su una macchina virtuale (ad esempio VirtualBox) con una qualsiasi versione di Windows da 3.1 fino a Windows Xp. Orientato al campo biomedico, include anche la statistica bayesiana (sensibilità, specificità, valore predittivo e curve ROC). Il file VB30.zip contiene il compilatore con il quale sono stati realizzati i file sorgente e l'eseguibile di Ministat.
